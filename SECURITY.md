# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the Metriqos C++ SDK, please report it responsibly.

**Do NOT open a public issue.** Instead, email us at:

**[security@metriqos.io](mailto:security@metriqos.io)**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Timeline

- **Acknowledgment:** within 48 hours
- **Assessment:** within 7 days
- **Fix:** critical issues patched within 14 days

## Scope

This policy covers the `metriqos-sdk-cpp` repository. For server-side vulnerabilities, contact the same email.

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest release | Yes |
| older releases | Best effort |
